Описание задания
Магазин должен позволять выбирать книги и делать покупки.

Сущности:
Книга, Книжный склад, Автор, Заказ, Детали заказа, Покупатель.
Книга содержит информацию о названии, идентификаторе автора, 
годе издания, количестве страниц, цене
Книжный склад хранит информацию об остатках книг по идентификатору 
книги (ИД у книг должны быть уникальными)
Автор - ФИО, ид
Покупатель - ид, ФИО, дата рождения
Заказ - ид покупателя, ид заказа, сумма покупки
Детали заказа - ид заказа, ид книги, количество, цена

Делаем слой энтити. Только сами энтити (дао для энтити делать не надо). 
Продумать систему связей между сущностями (будущие внешние ключи в бд).
